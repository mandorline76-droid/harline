# HarLine - Laporan Pemeliharaan Online ⚡

HarLine adalah aplikasi berbasis web yang dirancang khusus untuk memudahkan petugas lapangan dalam mendokumentasikan hasil pemeliharaan jaringan listrik (**HarTek**) dan pembersihan vegetasi (**HAR & ROW**).

## ✨ Fitur Utama
- **Dashboard Modern**: Ringkasan pekerjaan hari ini dengan tampilan premium.
- **Input HarTek**: Form input teknis lengkap dengan koordinat GPS otomatis.
- **Input HAR & ROW**: Form pembersihan vegetasi dengan dukungan foto (Sebelum/Sesudah).
- **Watermarking Foto**: Otomatis menambahkan label (Sebelum/Pelaksanaan/Sesudah) dan Timestamp pada foto.
- **Mode Offline**: (Planned) Dukungan penyimpanan draf saat sinyal lemah.
- **Desain Responsif**: Dioptimalkan untuk penggunaan di smartphone (Mobile-First).

## 🚀 Cara Penggunaan
1. Buka file `index.html` di browser (disarankan Chrome/Edge Mobile).
2. Login menggunakan akun petugas yang terdaftar.
3. Pilih menu **Input HarTek** atau **Input Harrow**.
4. Isi data lapangan dan ambil foto langsung dari kamera.
5. Klik **Kirim Laporan** untuk mengirim data ke sistem pusat.

## 🛠️ Teknologi
- **Frontend**: HTML5, Tailwind CSS, JavaScript (Vanilla).
- **Icons**: Google Material Symbols.
- **Backend**: Google Apps Script (Integration).
